诗歌鉴赏

分类类型

  1. 山水诗：李白、王维
  2. 送别诗：还是李白、王维
  3. 田园诗：陶渊明
  4. 边塞诗：南宋与金、南宋与元
  5. 象征诗：屈原

 

山水诗

李白：纵情山水

王维：信佛、看破红尘 => 安静的景 归隐闲适，动静结合、虚实结合

 

送别诗

李白：情深意厚

王维：情深意厚、仕途失意 => 思乡

 

山水田园诗

陶渊明：归隐；闲适

 

边塞诗

  1. 对边塞风光描写
  2. 战争的残酷、对统治者穷兵黩武的批判
  3. 思乡

南宋、金时期表达情感：

  1. 对国土大量沦陷的痛心和对金人的憎恨
  2. 对祖国的忠诚
  3. 为祖国建功立业，收复失地
  4. （对小人）苟且偷安、求和行径的不齿和失望

南宋、元表达情感（南宋灭亡）：

  1. 亡国之恨、故国之思
  2. 对故国忠诚
  3. 为故国建功立业、收复失地

 

象征诗

屈原：

  1. 君子：品行高洁、遭小人排挤、有才无法施展、内心痛苦

    香草、美人、大鹏、凤凰、高树、鸣蝉、蝴蝶

  2. 小人：排挤、得势

    恶草、众女、众鸟、燕雀、商风、落日余辉、浮云

感情表达

伤春悲秋：

  • 早春：开心
  • 仲春（清明):情感
  • 晚春：伤春

 

孤寂愁闷：

  • 感性：花开花落
  • 理性：仕途失意 => 可能反而会与民同乐、积极入世 （苏东坡）
  • 中性：对历史感兴趣 => 以史为鉴、借古讽今；今昔对比、时光易逝、历史充满沧桑感

 

十字口诀：

  • 喜：老百姓安居乐业
  • 怒：国仇家恨
  • 哀：早生华发
  • 乐：乐观的心态
  • 闲：闲适
  • 思：对远在外之人的思念、写对面的场景（落笔对面见深情）
  • 赞:咏物
  • 叹：悲叹
  • 愿：美好祝愿
  • 憾：遗憾、伤心
